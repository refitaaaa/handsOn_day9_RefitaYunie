Flutter Widget & Layouting

Latihan implementasi widget dan layout dasar Flutter

by Refita Yunie Samhuri

Daftar File Latihan

  • container_widget.dart — latihan membuat tampilan menggunakan Container
  • row_column_layout.dart — memahami layout Row dan Column
  • stack_positioned.dart — membuat tumpukan elemen dengan Stack dan Positioned
  • gridview_builder.dart — menampilkan daftar item dengan GridView.builder
  • listview_layout.dart — membuat daftar tampilan dinamis dengan ListView
  • form_input_widget.dart — latihan form input menggunakan TextField dan Button
